# AI Powered File Management System
This project allow you to create, read and update files with content by using Gmini LLM.

## Features
- Can Create any type of files. ex: .csv, .JSON, .txt.
- Can read and Update the file.
- Save user and AI response by using SQLite3 database.
- Modify the text using rich python library.

## Requirements

- Python 3.7 or later


## Installation

1. **Clone the repository:**

    ```
    https://github.com/ranwala/file_management_ai_agent.git
    cd file_management_ai_agent
    pip install -r requirements.txt
    ```

3. **How to run**
    ```
    python main.py
    ```

## Configuration
Create a .env file, and add below:
- GOOGLE_API_KEY=your_model_key
